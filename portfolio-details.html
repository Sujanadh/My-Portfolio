Summary of project

The precision farming project focused on the development of a web application that could predict moisture content in tomato plants using images of tomato leaves. The team collected and processed data using deep learning techniques to create a model, which was then deployed using Flask. The end result is a tool that helps farmers optimize their operations and make more informed decisions.
